## Build instructions
1. Install `npm`, `express`, and `cors`.
    ```
    sudo apt install npm
    npm install express cors
    ```
2. Start the server.
    ```
    cd *_server
    node server.js
    ```
3. Open the game.\
Open the `*_client` folder and double-click on `index.html`
4. Enjoy!
